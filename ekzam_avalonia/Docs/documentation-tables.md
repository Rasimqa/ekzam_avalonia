# Документация (табличный формат)

## 1. Сведения о разработке

| Параметр | Значение |
|---|---|
| Проект | Лавка «Матье» - управление услугами |
| Разработчик | Krast |
| Платформа | Windows 10/11, .NET 8, Avalonia 11 |
| Язык | C# |
| Архитектура | Code-behind, слои `Views`, `Services`, `Models` |

## 2. Сведения о сервере БД

| Параметр | Значение |
|---|---|
| СУБД | PostgreSQL 14+ |
| Провайдер | Npgsql |
| Подключение | `Services/DbSettings.cs` |
| Схема | `public` |
| Основные таблицы | `collections`, `services` |

## 3. Состав программных модулей

| Модуль | Назначение |
|---|---|
| `Views/LoginWindow` | Авторизация пользователя |
| `Views/MainWindow` | Просмотр, фильтрация, сортировка, пагинация, CRUD услуг |
| `Views/ServiceEditWindow` | Добавление/редактирование услуги |
| `Services/ServiceRepository` | SQL-операции чтения и изменения услуг |
| `Services/ServiceManager` | Бизнес-логика обновления даты изменения |

## 4. Реализованные функции

| Функция | Статус | Примечание |
|---|---|---|
| Постраничный вывод списка | Готово | 3 записи на странице |
| Сортировка по алфавиту | Готово | Кнопка `Сортировать А-Я` |
| Добавление услуги | Готово | Через `ServiceEditWindow` |
| Редактирование услуги | Готово | С обновлением `last_modified_date` |
| Вывод изображений | Готово | Загрузка через `avares://` + резервные изображения |
