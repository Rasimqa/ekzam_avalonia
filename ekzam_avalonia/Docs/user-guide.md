# Руководство пользователя

## 1. Назначение
Приложение предназначено для администрирования услуг лавки «Матье»: просмотр, добавление, редактирование, сортировка и постраничный вывод.

## 2. Роли
- **Администратор/оператор:** выполняет все функции текущей версии приложения.
- **Клиент:** в этой версии не имеет отдельного интерфейса, данные обслуживаются оператором.

## 3. Запуск
1. Подготовьте PostgreSQL.
2. Выполните скрипты из папки `Sql` в порядке:
   - `01_create_tables.sql`
   - `02_seed_data.sql`
3. Проверьте connection string в `Services/DbSettings.cs`.
4. Запустите проект `ekzam_avalonia`.

## 4. Интерфейс
- В верхней панели доступны кастомные кнопки:
  - `—` свернуть окно;
  - `X` закрыть окно (с подтверждением).
- Кнопки действий:
  - `Сортировать А-Я`
  - `Добавить`
  - `Редактировать`
- Ниже находится таблица услуг.
- Внизу расположены кнопки пагинации: `Назад` / `Вперед`.

## 5. Основные сценарии
### 5.1 Добавление услуги
1. Нажмите `Добавить`.
2. Заполните поля: название, цена, коллекция, картинка.
3. Нажмите `Сохранить`.
4. Услуга появится в таблице.

### 5.2 Редактирование услуги
1. Выберите строку в таблице.
2. Нажмите `Редактировать`.
3. Измените нужные поля.
4. Нажмите `Сохранить`.

### 5.3 Сортировка и страницы
- Нажмите `Сортировать А-Я`, чтобы включить сортировку.
- Используйте `Назад`/`Вперед` для перемещения между страницами (по 3 услуги).

### 5.4 Закрытие приложения
1. Нажмите `X`.
2. Подтвердите закрытие кнопкой `Да` или отмените кнопкой `Нет`.
