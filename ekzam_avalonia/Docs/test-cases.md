# Тест-кейсы (4-й блок, 5 шт.)

| № | Функция из 4 блока | Предусловие | Шаги | Ожидаемый результат | Фактический результат | Комментарий |
|---|---|---|---|---|---|---|
| TC-01 | Вывод информации в виде списка (постранично) | В таблице `services` больше 3 записей | Запустить приложение -> выполнить вход -> открыть главную форму | Отображается список карточек услуг, не более 3 на страницу, индикатор страницы корректный | Соответствует | Проверка UI и пагинации |
| TC-02 | Сортировка по алфавиту | В БД есть услуги с разными названиями | На главной форме нажать `Сортировать А-Я` | Карточки упорядочены по названию от А до Я | Соответствует | Проверка SQL сортировки |
| TC-03 | Редактирование данных выбранной записи | Есть минимум 1 услуга | Выбрать услугу -> `Редактировать` -> изменить цену -> `Сохранить` | Изменения отображаются в списке и записываются в БД | Соответствует | Проверка update-сценария |
| TC-04 | Фиксирование времени последнего изменения данных | Есть услуга для редактирования | Открыть редактирование -> изменить поле -> сохранить -> обновить список | Поле `last_modified_date` обновлено на текущее время | Соответствует | Критичный пункт задания |
| TC-05 | Добавление данных | Есть доступ к БД | Нажать `Добавить` -> заполнить форму -> `Сохранить` | Новая услуга появляется в списке и таблице `services` | Соответствует | Проверка create-сценария |
